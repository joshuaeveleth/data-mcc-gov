Millennium Challenge Corporation's Open Data Catalog
====================================================

Open Data Catalog for MCC.

Open Data initiative (@ProjectOpenData) - Presidential Innovation Fellows program.


Features
--------

* Publication of machine-readable data (e.g. XML, JSON, CSV/XLS, KML, ESRI Shapefile, etc)
* Publication of metadata
* References for non-machine-readable data (e.g. PDFs)
